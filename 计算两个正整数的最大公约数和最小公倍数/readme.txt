计算两个正整数的最大公约数和最小公倍数

接口说明
原型：
long getMaxDivisor(long lFirstInput, long lSecondInput);
输入参数：
      int first: 第一个整数;
      int second: 第二个整数;
返回值：
    最大公约数
 
long getMinMultiple(long lFirstInput, long lSecondInput);
输入参数：
      int first: 第一个整数;
      int second: 第二个整数;
返回值：
    最小公倍数

练习阶段:	 初级
